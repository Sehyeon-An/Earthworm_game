## 게임규칙

  * 지렁이는 현재 머리가 향하고 있는 방향으로 멈추지 않고 이동하지 합니다.
    플레이어의 조작을 통해서 머리의 진행 방향을 바꾸루 수 있습니다.
  
  * 지렁이는 벽이나 자신의 몸에 부딪히면 죽습니다.
  
  * 화면에 놓인 먹이를 먹으면 지렁이의 길이가 길어집니다. 
  
  * 화면의 먹이를 최대한 많이 먹어보세요!
  
  
  
## 조작방법

 * 이동 : 화살표키로 이동합니다.
 
 * 선택 : ENTER 키로 선택을 합니다.
